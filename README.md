# Spielanleitung – Tower of Hanoi

## Ziel des Spiels
Alle Scheiben sollen von einem Startstab auf einen anderen Stab verschoben werden.

## Spielaufbau
- 3 Stäbe
- mehrere unterschiedlich große Scheiben
- Start: alle Scheiben liegen auf einem Stab  
  (größte unten, kleinste oben)

## Regeln
1. Es darf nur **eine Scheibe gleichzeitig** bewegt werden.
2. Es darf nur die **oberste Scheibe** genommen werden.
3. Eine **größere Scheibe** darf niemals auf einer kleineren liegen.

## Spielablauf
1. Verschiebe die kleineren Scheiben auf den Hilfsstab.
2. Bewege die größte Scheibe auf den Zielstab.
3. Lege die kleineren Scheiben wieder auf die große Scheibe.

Diese Schritte werden wiederholt, bis der gesamte Turm auf dem Zielstab steht.

## Spielende
Das Spiel ist beendet, wenn alle Scheiben korrekt auf dem Zielstab aufgebaut sind.
